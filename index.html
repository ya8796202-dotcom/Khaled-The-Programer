تعلم أساسيات البرمجة

ما هي البرمجة؟

البرمجة هي عملية كتابة تعليمات لأجهزة الكمبيوتر لتنفيذ مهام محددة.

المبرمجون يستخدمون لغات برمجة مثل Python, Java, C++ لكتابة هذه التعليمات.

البرمجة تساعد في حل المشكلات وإنشاء تطبيقات ومواقع إلكترونية وألعاب وغيرها.

المتغيرات (Variables)

المتغيرات هي أماكن في الذاكرة لتخزين البيانات.

// في لغة JavaScript
let name = "أحمد"; // نص
let age = 25; // رقم صحيح
let height = 1.75; // رقم عشري
let isStudent = true; // قيمة منطقية

يمكن تغيير قيمة المتغير لاحقاً في البرنامج.

الشروط (Conditionals)

تستخدم الشروط لاتخاذ قرارات في البرنامج.

let age = 18;
if (age >= 18) {
    console.log("يمكنك التصويت");
} else {
    console.log("لا يمكنك التصويت بعد");
}

الحلقات التكرارية (Loops)

تستخدم الحلقات لتكرار مجموعة من التعليمات.

// مثال لحلقة for
for (let i = 0; i < 5; i++) {
    console.log("مرحباً " + i);
}

// مثال لحلقة while
let count = 0;
while (count < 5) {
    console.log("العدد: " + count);
    count++;
}

الدوال (Functions)

الدوال تساعد في تنظيم الكود وإعادة استخدامه.

function greet(name) {
    return "مرحباً " + name;
}

let message = greet("أحمد");
console.log(message); // يطبع: مرحباً أحمد

مقدمة في التشفير

ما هو التشفير؟

التشفير هو عملية تحويل المعلومات إلى شكل غير مفهوم لحماية سريتها.

أنواع التشفير:

  • التشفير المتماثل: يستخدم نفس المفتاح للتشفير وفك التشفير
  • التشفير غير المتماثل: يستخدم مفتاحين (عام وخاص)

التشفير يحمي بياناتك أثناء نقلها عبر الإنترنت ويحمي خصوصيتك.

أهمية كلمات السر القوية

كلمات السر القوية تحمي حساباتك من الاختراق والوصول غير المصرح به.

مواصفات كلمة السر القوية:

  • طول لا يقل عن 12 حرفاً
  • تحتوي على أحرف كبيرة وصغيرة
  • تحتوي على أرقام ورموز خاصة
  • لا تحتوي على معلومات شخصية يسهل تخمينها
  • مختلفة لكل حساب

استخدم مدير كلمات السر لتذكر كلمات سر قوية وفريدة لكل موقع.
